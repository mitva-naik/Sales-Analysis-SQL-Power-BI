# Sales-Analysis-SQL-Power-BI
Sales performance analysis using SQL and Power BI (Kaggle dataset)


##  Project Overview
This project analyzes sales performance using a real-world Superstore sales dataset sourced from Kaggle.  
SQL was used for data analysis, and Power BI was used to build an interactive dashboard.

##  Dataset
- Source: Kaggle (Superstore Sales Dataset)

##  Tools Used
- SQL (MySQL)
- Power BI
- CSV Dataset

##  Business Questions Answered
- Which regions and categories generate the highest sales?
- Who are the top customers by revenue?
- What are the monthly sales trends?
- How do sales vary by segment and shipping mode?
